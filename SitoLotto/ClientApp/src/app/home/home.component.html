Welcome to my little website

This site was designed and built during the March 2020 quarantine caused by the Covid - 19 virus. the initial idea was to create an environment on which to test new technologies, in free time (which was abundant at that time). Now it has also become a practical site where I have developed useful tools, such as the comparison of web pages.

The Solution of this site is available on the following GIT Repository Repository The Solution has the following projects and libraries

  • LbExcel

    Library for creating and reading Excel files created with the Framework .NET Standard2.0

  • TestLibrerie

    Test project of the libraries present in the "Generic libraries" folder created with the FrameworkASP.NET Core

  • libraryLotto

    Library for managing the loading and use of data relating to SuperEnalotto.NET Standard2.0

  • LottoUnitTest

    Test project of the libraries present in the "Generic libraries" folder created with the Framework ASP.NET Core

  • SitoLotto

    Project containing the sources of the site in question created with the FrameworkASP.NET Core

Technologies and frameworks used

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
    • documentformat.openxml
  • Bootstrap for layout and styling
  • OpenXmlTo read and write Excel files
  • Kendo For User Interface
  • DiffPlexTo compare find differences between strings and files
  • FluentScheduler For scheduling asynchronous tasks
